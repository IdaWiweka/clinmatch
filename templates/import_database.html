Import Database

Use this form to import data from a JSONL file into the database.

Important information:

  • Each record in the JSONL file must have text and text_id fields.
  • Categories and entities should be additional fields in each record.
  • Example format: {"text_id":"1", "text":"Patient has fever", "symptoms":["fever", "cough"]}
  • Existing texts with the same text_id will be identified but not overwritten.
